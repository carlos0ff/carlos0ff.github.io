Sobre o Projeto

O sistema de aluguel de carros foi desenvolvido como projeto acadêmico, com foco na criação de um painel administrativo funcional para o gerenciamento de clientes, veículos, reservas e controle de disponibilidade em tempo real.

Utilizando PHP para o backend e MySQL para o banco de dados, a aplicação permite cadastrar, editar e remover registros, realizar buscas dinâmicas e emitir comprovantes de locação.

Gestão de Veículos

Cadastro, edição, exclusão e status de disponibilidade em tempo real.

Clientes

Gerenciamento completo de clientes com CPF e histórico de locações.

Reservas

Controle de datas, horários, veículo reservado e emissão de recibos.

Tecnologias

PHP procedural, MySQL, HTML/CSS, e boas práticas de segurança básica.

1 Consulta de Clientes e Contratos Recuperar o nome e telefone dos clientes, número de contrato, modelo e placa do veículo. Ordenado pela data de saída mais recente
SQL 

      SELECT 
          c.nome AS NomeCliente,
          t.numero AS Telefone,
          ct.numero AS NumeroContrato,
          v.modelo AS ModeloVeiculo,
          v.placa AS PlacaVeiculo,
          ct.dataSaida AS DataSaida
      FROM 
          cliente c
      LEFT JOIN 
          telefone t ON c.codigo = t.codigoCliente
      JOIN 
          contrato ct ON c.codigo = ct.codigoCliente
      JOIN 
          veiculo v ON ct.idVeiculo = v.id
      ORDER BY 
          ct.dataSaida DESC;
