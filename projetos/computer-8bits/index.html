Sobre o Projeto

Este computador de 8 bits é uma implementação física completa de uma arquitetura de computador simples, construída com componentes discretos em protoboard. O projeto inclui CPU, memória RAM, barramentos de dados e controle, e um conjunto de instruções personalizado capaz de executar operações básicas.

Arquitetura

Design baseado no modelo de Von Neumann com barramento único.

Memória

16 bytes de RAM implementados com flip-flops.

Clock

Clock ajustável de 1Hz a 10Hz para depuração.

Instruções

Conjunto de 12 instruções incluindo operações aritméticas.

Detalhes Técnicos

Arquitetura

O computador implementa uma arquitetura de Von Neumann com um barramento compartilhado para dados e instruções. A CPU contém uma ALU simples capaz de realizar operações aritméticas e lógicas básicas, dois registradores de propósito geral (A e B), e um contador de programa.

Componentes Principais

74LS181 (ALU) 74LS173 (Registradores) 74LS189 (RAM) 74LS161 (Contador) 74LS245 (Barramento) 74LS139 (Decodificador)

Conjunto de Instruções

Opcode Instrução Descrição
0000 NOP Nenhuma operação
0001 LDA [addr] Carrega valor em A
0010 LDB [addr] Carrega valor em B
0011 STA [addr] Armazena A na memória
0100 ADD Soma A e B (resultado em A)
0101 SUB Subtrai B de A (resultado em A)
0110 JMP [addr] Salto incondicional
